Fine-mapping of cancer GWAS

Fine-mapping was done on several cancer types. Prior weights were obtained from TORUS. I selected all LD blocks that have an FDR \(\leq 10\)% based on the q-values from TORUS. I used L = 1 for fine-mapping with susie_rss() which works on out-of-sample LD.

The SNP covariance matrix was estimated as follows as adviced by Yuxin:

\[ R = (X^TX + \hat{z}\hat{z}^T)/n\]

where \(X\) is the \(n \times p\) 1000 Genomes genotype matrix, which has been column centered and scaled to unit variance. The outer-product of z-scores \(\hat{z}\hat{z}^T\) penalize the covariance when the signs of \(\hat{z}\) dont agree.

To get the LD correlation matrix \(r_{ij}\), each element of \(R\) is transformed as follows:

\[ r_{ij} = \frac{R_{ij}}{\sqrt{R_{ii}}\sqrt{R_{jj}}} \]

Add functional annotation used in Torus

top.snps <- merged.sumstats[merged.sumstats$susie_pip > 0.1 & merged.sumstats$CS == 1, ]

annot.df <- suppressMessages(vroom::vroom('~/GERMLINE/SUMMARY_STATISTICS/torus/BRCA_Torus_annotations.txt.gz', col_names = T))

top.snps <- inner_join(top.snps, annot.df[ ,c("snp","merged")], by = "snp")
